# 多态（Polymorphism）

## 1. 对象的实际类型由 `new` 决定

在 Java 中，通过 `new` 创建对象时，对象的**实际类型**由 `new` 后面的类决定：

```java
Animal a = new Dog();
```

- 变量 `a` 的**编译期类型**是 `Animal`（左边的类型）。
- 因为编译期类型是 `Animal`，所以 `a` **只能调用 `Animal` 中声明的方法**（`sound()`），不能直接调用只在 `Dog` 中存在的 `bark()`。
- 对象的**运行期实际类型**是 `Dog`（`new` 右边的类型）。
- 如果 `Dog` 重写了 `Animal` 的 `sound()` 方法，那么 `a.sound()` **在运行时执行的是 `Dog` 的实现**。

多态的关键点：**引用类型看左边，实际对象看右边；方法调用的实际执行版本取决于右边（运行时类型）。**
